## 1. 과제 개요

**개별 과제 설명**

이 과제는 여러 장의 이미지를 순차적으로 전환하며 표시할 수 있는 웹 기반 이미지 캐러셀(Carousel) 애플리케이션을 제작하는 것으로, 좌우 버튼·자동 슬라이드·전환 애니메이션·인디케이터 등 다양한 조작 기능을 구현하여 제한된 공간에서도 시각적으로 효율적인 콘텐츠 표현이 가능하도록 구현해야 합니다.

## 2. 요구 사항 및 유의 사항

### 요구사항

아래 제시된 핵심 기능을 개발합니다.

**1. 기본 기능**

- 여러 장의 이미지를 API 요청을 통해 불러와 순차적으로 표시하는 이미지 캐러셀 구현

- 마우스 클릭을 통한 이전(◀) / 다음(▶) 이미지 이동 기능

- 키보드 방향키( ← / → )를 통한 이미지 이동 기능

- 현재 이미지 위치를 나타내는 페이지 인디케이터(●) 표시 기능

- 이미지 전환 시 적용되는 부드러운 전환 애니메이션(transition) 제공 기능

**2. 자동 재생 기능 (Auto Slide)**

- 일정 시간 간격(예: 3초)으로 다음 이미지로 이동하는 자동 재생 기능

- 자동 재생 중 Auto 버튼 클릭 시 자동 재생을 일시정지하는 Pause 기능

- 일시정지 이후 자동 재생을 다시 시작하는 재시작 기능

### 유의 사항

**캐러셀·애니메이션 핵심 로직과 직접적으로 관련된 라이브러리는 사용 불가능합니다. swiper, react-slick, embla-carousel, keen-slider 등 외부 라 이브러리 사용 금지**

## 3. 기술 및 환경 설정

### API/데이터 소스

json-server를 사용하여 RESTful API를 제공합니다.

#### json-server 설정

1. json-server 설치

```bash
npm install -g json-server

# 또는

pnpm add -g json-server
```

2. json-server 실행

```bash
json-server --watch db.json --port 3002
```

#### API 엔드포인트

**기본 URL:** `<http://localhost:3002>`

**날짜 목록 조회**

- **URL:** GET /images

- **Response:**

```typescript
[
  {
    id: "0",
    author: "Alejandro Escamilla",
    url: "https://picsum.photos/id/0/800/600",
  },
];
```

## api 요청을 통해 순차적으로 표시하는 이미지 캐러셀 구현

### 순차적으로 보여주는 컨테이너 제작

리액트 쿼리를 사용하여 이미지를 불러오고 이미지배열에 flex를 적용하여 일자로 보이게 만들었습니다 그 후 캐러셀컨테이너에 overflow hidden을 적용하여 상자를 넘는 요소를 감추었습니다.

해당 캐러셀은 컴포넌트화하여 사용자가 재사용할 수 있게끔 아이템을 감싸는 컴포넌트와 실제 캐러셀이 이루어지는 컴포넌트로 나누어 구현했습니다.

### 버튼제작

버튼은 사용자가 커스텀한 버튼을 받을 수 있도록 하였고 다음 이전 버튼을 누를 시 해당 캐러셀의 너비만큼 트랜스레이트하게 만들었습니다.

### 키보드 이벤트 제작

키보드를 입력할 시 에드리스너를 통해 함수를 실행하고 해당 키를 판별후 함수를 실행하여 구현했습니다.
종속성을 추가하여 업데이트된 인덱스값이 적용되도록 하였습니다.

### 현재 위치를 나타내는 인터케이터 표시기능제작

이미지의 총 너비를 기준으로 인디케이터를 표시하고 현재 인덱스에 따라 색이 변하도록 하였습니다.

### 드래그기능

클릭하여 드래그시 기준에서 벗어나는 길이만큼 움직이고 캐러셀의 너비를 벗어나지 않게 하였습니다.
그 후 마우스를 올렸을 때 움직인 값이 너비의 20%를 넘기고 음수이면 이전 양수이면 이후의 이미지를 보여주게 하고 그외의 경우는 원래의 index를 보여주도록 하였습니다.

### 자동재생기능

셋인터벌을 사용해 3초마다 한번씩 next함수를 실행하도록하고 index가 최대길이 도달시 0으로 초기화 되도록 만들었습니다.

자동재생버튼은 사용자가 커스텀후 사용할 수 있도록 변수로 컨트롤 하였습니다.
